# 茶颜悦色·桂花弄 H5推广页面

这是一个完整的移动端H5产品推广页面，展示茶颜悦色秋季限定新品"桂花弄"奶茶。

## 功能特点

### 页面内容（共8页）
1. **开场动画页** - 桂花飘落动画 + 品牌LOGO
2. **产品展示页** - 产品图片展示 + 购买按钮
3. **原料溯源页** - 产地地图展示 + 信息卡片
4. **产品成分页** - 一镜到底制作动画 + 成分介绍
5. **互动游戏页** - 摇一摇砸金蛋赢优惠券
6. **用户评价页** - 滚动展示用户好评
7. **门店地图页** - 门店位置 + 导航功能
8. **结束引导页** - 二维码关注 + 分享功能

### 动画效果
- ✨ Canvas实现的桂花飘落粒子动画
- 🌊 页面一镜到底滑动切换效果
- 💫 元素渐显、缩放、旋转等多种动画
- 🎯 点击水波纹、呼吸灯光等交互反馈

### 互动功能
- 📱 支持触摸滑动切换页面
- 🎮 摇一摇或点击砸金蛋游戏
- 🎵 背景音乐播放控制
- 💬 Toast提示系统
- 📦 成分详情弹窗

## 使用方法

### 1. 准备素材
在项目根目录创建以下文件夹并添加素材：

**images文件夹：**
- `logo.png` - 茶颜悦色品牌LOGO（建议尺寸：300x300px）

**music文件夹：**
- `bg-music.mp3` - 背景音乐（建议使用古筝轻音乐，2分钟左右）

### 2. 本地测试

**方法一：使用VS Code Live Server**
1. 安装VS Code编辑器
2. 安装Live Server插件
3. 右键点击`index.html` → "Open with Live Server"
4. 浏览器会自动打开，使用手机模式查看（F12开发者工具）

**方法二：直接打开**
1. 双击`index.html`文件
2. 在浏览器中按F12打开开发者工具
3. 切换到移动设备模式查看效果

### 3. 手机测试
1. 将项目部署到Web服务器
2. 或使用本地服务器（如Python: `python -m http.server 8000`）
3. 手机与电脑在同一网络下访问电脑IP地址

### 4. 线上部署

**使用H5制作平台（推荐）：**
- 意派Epub360 (https://www.epub360.com/)
- 兔展 (https://www.rabbitpre.com/)
- 易企秀 (https://www.eqxiu.com/)

将HTML、CSS、JS代码导入平台，上传素材即可生成二维码和链接。

**自行部署：**
1. 上传到自己的服务器
2. 或使用GitHub Pages免费托管
3. 或使用Netlify、Vercel等免费服务

## 操作说明

### 页面切换
- **触摸滑动**：向上滑动进入下一页，向下滑动返回上一页
- **鼠标滚轮**：（桌面测试用）滚轮向下到下一页，向上返回上一页

### 音乐控制
- 点击右上角音乐图标🎵可以暂停/播放背景音乐

### 砸金蛋游戏
1. 进入第5页
2. 点击"开始游戏"按钮
3. 摇动手机或直接点击金蛋
4. 每人每天可以砸3个金蛋

### 成分查看
- 在第4页点击任意成分可以查看详细介绍

## 技术特点

### 响应式设计
- 使用rem单位实现多尺寸适配
- viewport设置禁止缩放
- 针对不同屏幕尺寸的媒体查询

### 性能优化
- CSS3动画替代JavaScript动画
- Canvas粒子数量控制
- 事件节流防止过度触发

### 兼容性
- 支持iOS Safari
- 支持Android Chrome
- 支持微信内置浏览器

## 自定义修改

### 修改颜色
在`style.css`文件顶部的`:root`部分修改颜色变量：
```css
:root {
    --primary-color: #D4A574;  /* 主色调 */
    --dark-brown: #8B4513;     /* 深棕色 */
    --light-beige: #FFE4B5;    /* 浅杏色 */
    --chinese-red: #DC143C;    /* 中国红 */
}
```

### 修改文案
直接在`index.html`中查找对应的文字内容进行修改。

### 修改动画速度
在`style.css`中找到对应的`animation`属性，修改时长参数。

### 添加更多页面
1. 在`index.html`中复制一个`<div class="page">`结构
2. 修改`data-page`属性为新的页码
3. 在`script.js`中修改`totalPages`变量

## 注意事项

⚠️ **重要提示：**
1. 部分浏览器需要用户交互后才能播放音频
2. 摇一摇功能需要HTTPS环境才能正常使用
3. 地图功能需要接入百度/高德地图API
4. 二维码需要生成真实的公众号/小程序码
5. 图片素材建议使用WebP格式以提升加载速度
