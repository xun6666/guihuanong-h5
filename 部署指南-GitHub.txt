===============================================
使用 GitHub Pages 部署（永久免费）
===============================================

第一步：准备 GitHub 账号
--------------------------
1. 访问 https://github.com/
2. 注册账号（如果还没有）
3. 登录

第二步：创建仓库
--------------------------
1. 点击右上角 "+" → "New repository"
2. 填写信息：
   Repository name: guihuanong-h5
   Description: 茶颜悦色桂花弄H5推广页面
   选择 "Public"（公开）
3. 点击 "Create repository"

第三步：上传文件
--------------------------
方法A - 网页上传（简单）：
1. 在仓库页面，点击 "uploading an existing file"
2. 将所有文件拖拽到页面（index.html, style.css, script.js, images文件夹, music文件夹等）
3. 点击 "Commit changes"

方法B - 使用 GitHub Desktop（推荐）：
1. 下载安装 GitHub Desktop
2. Clone 刚创建的仓库
3. 将所有项目文件复制到仓库文件夹
4. Commit 并 Push

第四步：启用 GitHub Pages
--------------------------
1. 在仓库页面，点击 "Settings"
2. 左侧菜单找到 "Pages"
3. 在 "Source" 下拉菜单中选择 "main" 分支
4. 点击 "Save"
5. 等待1-2分钟

第五步：访问网站
--------------------------
地址格式：https://你的用户名.github.io/guihuanong-h5/

例如：
- 用户名是 zhangsan
- 访问地址：https://zhangsan.github.io/guihuanong-h5/

第六步：生成二维码
--------------------------
1. 复制上面的访问地址
2. 访问：https://cli.im/（草料二维码）
3. 粘贴链接，生成二维码
4. 下载保存

===============================================
优点：
✓ 完全免费
✓ 永久有效
✓ 支持自定义域名
✓ 访问速度快（有CDN）
✓ 可以随时更新内容

更新内容：
只需要修改文件后，重新上传或Push，网站会自动更新！
